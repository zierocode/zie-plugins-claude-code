# zie-plugins-claude-code

Claude Code plugin marketplace for [zierocode](https://github.com/zierocode) tools.

## Plugins

| Plugin | Description |
|--------|-------------|
| [zie-framework](https://github.com/zierocode/zie-framework) | Solo developer SDLC framework — spec-first, TDD, automated testing |

## Install

```bash
# Add this marketplace
claude plugin marketplace add https://github.com/zierocode/zie-plugins-claude-code.git

# Install a plugin
claude plugin install zie-framework@zie-plugins
```
